# KI Website

The KI Website is the public publication for Knowledge Islands at [knowledgeislands.info](https://knowledgeislands.info). It turns selected philosophy, model, and practical tooling material into an accessible, independently deployable site.

## Place in the Knowledge Islands ecosystem

The website is not a canonical source of philosophy, tooling, or specifications. It vendors source-labelled material from [Arcadia Principal](https://github.com/knowledgeislands/ki-arcadia-principal), which owns the Knowledge Islands philosophy and model; from the [KI Agentic Harness](https://github.com/knowledgeislands/ki-agentic-harness), which owns reusable agentic tooling; and from [KI Specifications](https://github.com/knowledgeislands/ki-specifications), which owns normative portable contracts.

That vendor model lets the website build and deploy autonomously while canonical ownership remains with each source repository. The mirrored [ecosystem decision](docs/decisions/GDR-KI-ARCADIA-002-knowledge-islands-ecosystem-fundamentals.md) defines the four authorities and publication flows, and the site provides stable public routes for [user installation](https://knowledgeislands.info/harness/install) and [repository bootstrap](https://knowledgeislands.info/harness/bootstrap).
