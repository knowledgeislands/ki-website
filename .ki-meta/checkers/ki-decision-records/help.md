# ki-decision-records

Codify, audit, and maintain Decision Records in any Knowledge Islands repo — the unified instrument replacing ki-adrs and ki-kdrs.

**Invoke:** `ki-decision-records audit [dir] | conform [dir] | help | educate [dir] | new <scope> "<title>" | refresh`

**Modes:**

- `AUDIT`
- `CONFORM`
- `EDUCATE`
- `HELP` — explain this skill and stop; the default when no mode is given (then routes, if interactive)
- `NEW`
- `REFRESH`

**See also:** ki-kb (island structure and frontmatter standard), ki-kb-streams (Enactment Process).
